Mountain Summit Book (for Android)

The mountain summit book (android) provides an app running on android with the possibility to create a log containing all mountain summit you've been on. Multimedia-style.

Download this project as a .zip file Download this project as a tar.gz file

Internal Open Source Project of ERNI Community

Welcome to the Mountain Summit Book (for Android) project

Features

  • Post a new summit log with geolocation
  • Add descriptions to your new entry
  • (in progress) Add a photo to your new entry
  • (planned) Publish your achievement on social media
  • (planned) See others achievements
  • (planned) See others log entries for the same mountain summit

Requirements

  • Android 2.2 or higher
  • Device supporting GPS
  • Internet access (to use community features)
